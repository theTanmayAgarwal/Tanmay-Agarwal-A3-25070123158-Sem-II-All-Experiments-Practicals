## ğŸ“˜ Theory

### ğŸ”¹ Operations on Set

#### 1. Creating a Set
A set is created using curly brackets `{}`.
Example: `my_set = {1, 2, 3, 4}`

#### 2. Adding Elements
Elements can be added using `add()` method.
Example: `my_set.add(5)`

#### 3. Removing Elements
Elements can be removed using `remove()` or `discard()`.
Example: `my_set.remove(2)`

#### 4. Set Operations
* Union
* Intersection
* Difference
* Symmetric Difference

#### 5. Set Methods
* `add()`
* `remove()`
* `discard()`
* `pop()`
* `clear()`

---

### ğŸ”¹ Operations on Dictionary

#### 1. Creating a Dictionary
A dictionary is created using curly brackets with keyâ€“value pairs.
Example: `my_dict = {"name": "Tanmay", "age": 18}`

#### 2. Accessing Values
Values are accessed using keys.
Example: `my_dict["name"]`

#### 3. Updating Values
Values can be updated using keys.
Example: `my_dict["age"] = 19`

#### 4. Removing Elements
Elements can be removed using `pop()` or `del`.
Example: `del my_dict["age"]`

#### 5. Dictionary Methods
* `keys()`
* `values()`
* `items()`
* `get()`
* `update()`
