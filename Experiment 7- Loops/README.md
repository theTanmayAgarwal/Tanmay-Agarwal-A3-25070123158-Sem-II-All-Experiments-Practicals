## ğŸ“˜ Theory (Loops in Python)

### 1. While Loop

A **while loop** executes a block of code as long as a given condition is true.

**Syntax:**  
`while condition:`  
&nbsp;&nbsp;&nbsp;&nbsp;`statement`

**Example:**  
`i = 1`  
`while i <= 5:`  
&nbsp;&nbsp;&nbsp;&nbsp;`print(i)`  
&nbsp;&nbsp;&nbsp;&nbsp;`i += 1`

---

### 2. For Loop

A **for loop** is used to iterate over a sequence such as a list, tuple, or range.

**Syntax:**  
`for variable in sequence:`  
&nbsp;&nbsp;&nbsp;&nbsp;`statement`

**Example:**  
`for i in range(1, 6):`  
&nbsp;&nbsp;&nbsp;&nbsp;`print(i)`

---

### 3. Nested Loop

A **nested loop** is a loop inside another loop.  
The inner loop executes completely for each iteration of the outer loop.

**Example:**  
`for i in range(1, 4):`  
&nbsp;&nbsp;&nbsp;&nbsp;`for j in range(1, 4):`  
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;`print(i, j)`
