## ğŸ“˜ Theory (Operations on Tuple)

### 1. Creating a Tuple

A tuple is created using round brackets ().
Example: `my_tuple = (1, 2, 3, 4)`

### 2. Accessing Elements

Elements are accessed using index numbers.
Example: `my_tuple[0]`

### 3. Tuple Immutability

Tuple elements cannot be changed once created.
Example: `my_tuple[1] = 10` (Error)

### 4. Tuple Operations

Example: `(1,2) * 2`
